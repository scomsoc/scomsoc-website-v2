## Hugo Directory Structure

It will be very useful to understand Hugo's directory structure from their [documentation](https://gohugo.io/getting-started/directory-structure/). I have added some content in the `content` folder which should help you in following the directory structure. The `config` folder is where you would make overall changes to the site, such as logo, menu options, etc. There is also a way to override the styling, I haven't figured it out, but Blowfish [documentation](https://blowfish.page/docs/homepage-layout/#custom-layout) should be of help.
